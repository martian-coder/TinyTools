01The problem

Every current solution reads the child's messages.

Commercial parental-control products upload children's private conversations to vendor clouds for scanning — creating a permanent surveillance record, a breach liability, and a data-protection problem no public institution can responsibly endorse. The alternative offered to parents has been binary: total surveillance, or none.

Perch rejects the premise. Detection is a local computation: it can run where the data already is — on the phone — and transmit only its conclusion.

02Architecture

Data minimization enforced by design, not policy.

Transmitted on detection — the complete list

  • Threat category and severity
  • Plain-language reason (from a fixed set)
  • Source app label
  • sender display name
  • timestamp

Never transmitted, never stored off-device

  • Message content — the events table has no content column
  • Media, photos, audio
  • Conversation history or contact lists
  • Clean traffic — non-flagged notifications produce no record off-device
  • Location, browsing, app usage
03Detection capabilities

Eight threat classes, tuned for how offenders actually write.

04Operating principles

Built to be defensible — to parliaments, to press, to children.

P-1

Transparency to the child

Setup runs on the child's device and explains, in age-appropriate language, exactly what is monitored. The child can always view every flag ever raised — the same list the guardian sees.

P-2

Data minimization by construction

The reporting schema physically cannot carry message content. Data protection is a property of the architecture, aligned with GDPR minimization principles — not a promise in a privacy policy.

P-3

Independent verifiability

The detection engine, relay, clients and this page are published source code. Any auditor, journalist or ministry can verify every claim on this page against the implementation.

P-4

Offline operation

Detection requires no connectivity and no third-party AI service. The device analyzes its own traffic; the network carries conclusions only.
