Quick Start
1. Set your context
Before joining a call, paste your agenda, role, and talking points into the Context box on the Start tab. This becomes your cached system prompt — the AI uses it to tailor every response.
2. Choose a profile
Pick the profile that matches your session: Meeting for team discussions, Sales for client calls, Presentation for pitches, or General for everything else.
3. Start & switch to your call
Hit ⌘↵ or click Start Session, then switch to Zoom / Meet. The transparent overlay floats on top. When the other person finishes speaking, a suggested response streams automatically.

System Audio Setup
macOS: Install BlackHole 2ch, create a Multi-Output Device in Audio MIDI Setup (Built-in + BlackHole), and set it as your output. Works perfectly with any headphones.

Windows: Enable "Stereo Mix" in Sound settings. Works natively with WASAPI loopback.

Linux: Set your PulseAudio monitor source. Run pactl list sources to find it.
