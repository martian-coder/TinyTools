# Deploy Strenes in 30 Seconds 🚀

Your app is built and ready! Choose the easiest option:

---

## ⚡ Option 1: Vercel (Recommended - 1 minute)

**Easiest - No code needed, just click buttons!**

1. Go to: https://vercel.com/new
2. Click **Import Git Repository**
3. Paste: `https://github.com/martian-coder/tinytools`
4. Click **Import**
5. Set **Root Directory** to `Strenes`
6. Click **Deploy**
7. **Done!** You get a live URL 🎉

That's it! Vercel will auto-build and deploy.

---

## 🌐 Option 2: Netlify Drop (No Git Needed)

**Super easy - Just drag & drop!**

1. Go to: https://app.netlify.com/drop
2. Drag the `Strenes/dist` folder into the browser
3. **Done!** You get a live URL 🎉

---

## 📦 Option 3: GitHub Pages

**If you have GitHub account:**

1. Create a new repo `strenes-app`
2. Upload the `dist` folder
3. Go to **Settings** → **Pages**
4. Select `dist` folder as source
5. **Done!** URL is shown 🎉

---

## Which One?

**On Your Phone Right Now?**
→ Try **Option 2 (Netlify Drop)** - super simple, just drag folder

**At Your PC Later?**
→ Try **Option 1 (Vercel)** - best overall, auto-deploys
