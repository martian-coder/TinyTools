# 🚀 JumpDir

> Interactive fuzzy directory navigator — type `j` and teleport anywhere on your filesystem.

Like `cd` but with fuzzy search, frecency ranking, and a TUI that actually sparks joy.

---

## ✨ Features

- 🔍 **Fuzzy matching** — Type partial names, it finds the right folder
- 📈 **Frecency ranking** — Frequently + recently visited directories bubble to the top
- 🌐 **Multi-scope search** — Current directory, specific drive, or all drives
- 🎨 **Beautiful TUI** — ANSI-colored interactive picker with keyboard navigation
- 🖥️ **Cross-platform** — Windows (CMD/PowerShell), macOS, Linux (bash/zsh)
- 📦 **Zero dependencies** — Pure Python 3 standard library, nothing to `pip install`

---

## 🚀 Quick Start

### Windows (CMD / PowerShell)
```bash
# One-time setup — creates j.bat
python jumpdir.py --make-bat

# Add the JumpDir folder to your PATH, then:
j              # Open scope picker → fuzzy search
j proj         # Search current directory for "proj"
j -d proj      # Search entire D: drive
j -g proj      # Search all drives (global)
```

### macOS / Linux (bash)
```bash
# One-time setup
echo 'eval "$(python3 jumpdir.py --init)"' >> ~/.bashrc
source ~/.bashrc

j              # Interactive picker
j azure        # Pre-filled search
```

### macOS / Linux (zsh)
```bash
echo 'eval "$(python3 jumpdir.py --init-zsh)"' >> ~/.zshrc
source ~/.zshrc
```

## 🎯 How It Works

1. **Scope Selection** — Pick where to search (local tree, specific drive, or global)
2. **BFS Discovery** — Scans directories breadth-first, skipping noise (`node_modules`, `.git`, `Windows`, etc.)
3. **Fuzzy Match** — Scores each directory name against your query using substring + character-sequence matching
4. **Frecency Boost** — Directories you visit often and recently get ranked higher
5. **TUI Picker** — Arrow keys to navigate, Enter to jump, Esc to cancel

The selected directory path is emitted to stdout, and the shell wrapper `cd`s into it.

---

## 🏗️ Architecture

```
jumpdir.py     → Everything: TUI, fuzzy matcher, BFS scanner, frecency DB, shell integration
j.bat          → Windows CMD wrapper (generated by --make-bat)
install.sh     → Unix one-line installer for bash/zsh
```

### Frecency Database

JumpDir maintains a tiny JSON database at `~/.jumpdir_db.json` that tracks:
- **Visit count** per directory
- **Last visit timestamp**

The ranking formula: `fuzzy_score + (visits × 3) + max(0, 20 - age_in_days × 2)`

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate results |
| `Enter` | Jump to selected directory |
| `Esc` | Cancel |
| `Backspace` | Delete last character |
| `Delete` | Clear query |
| Any character | Append to search query |
